# #11 [Story] ユーザーはタスクにラベルを付けられる

- **State**: OPEN
- **Labels**: user-story
- **Created**: 2026-03-13

---

## Why
**As** タスク管理者
**I want** タスクにカテゴリラベルを付けたい
**So that** タスクを種類別に分類・フィルタリングできるからだ

## Acceptance Criteria

```gherkin
Scenario: ラベルの追加
Given: タスク作成/編集モーダルが表示されている
When: ラベル「バグ」を追加して保存した時
Then: タスクカードにラベルバッジが表示される
```

```gherkin
Scenario: ラベルでフィルタリング
Given: ラベル付きタスクがある
When: フィルターでラベル「機能追加」を選択した時
Then: そのラベルのタスクのみ表示される
```

## Notes
- プリセットラベル: バグ, 機能追加, 改善, ドキュメント
- カスタムラベルも作成可能
- ラベルは複数付与可能
